# Инструкция к MarkDown

## Выделение текста

Для выделения курсивом обромить текст *

Для выделения полужирным обрамить текст **

Для выделения зачеркнутого текста обрамить его ~~

*Жили были у бабуси*

**Два веселых гуся**

~~Один серый, один белый~~

**Два веселых гуся**
